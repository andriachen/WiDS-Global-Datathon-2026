## 📌 Overview

This repository contains our submission for the [WiDS (Women in Data Science) Global Datathon 2026](https://www.widsconference.org/datathon.html), which focused on predicting wildfire threat levels across different geographic regions. Our model achieved a leaderboard score of **0.97834**, third place as of March 1st, 2026! This was an improvement from our first submission with a score of **0.96588**.

The goal of the competition was to build a machine learning model that accurately predicts the threat level of wildfires given environmental and geographic features, helping inform disaster preparedness and resource allocation efforts.
